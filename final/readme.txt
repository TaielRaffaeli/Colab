Es un proyecto de aprendizaje automatico, con una base de datos del clima de Australia, lo cual ensayamos modelos para predicciones.
Para ejecutarla se recomienda mediante un entorno virtual.
requerimientos de ejecucion:
*streamlit
instrucciones:
abrir la consola y cambiar de directorio hasta donde se encuentren los archivos app_clima.py y clima_pipeline.joblib.
introducir el comando: streamlit run .\app_clima.py
se abrira una ventana del navegador donde se interactua con la prediccion.
elegir los datos a predecir y presionar el boton para esperar el resultado, donde 1 es llueve y 0 no llueve.
